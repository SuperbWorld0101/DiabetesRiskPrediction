# DiabetesRiskPrediction
a new ML project to predict diabetes risk by analyzing relevant factors
After preprocessing and converting the continuous diabetes risk score into a binary classification problem, the logistic regression model achieved 95% accuracy on the test set. The dataset is balanced, so this performance indicates strong predictive power. Logistic Regression was selected as a baseline due to its interpretability and established use in medical risk modeling.
